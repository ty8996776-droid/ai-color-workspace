# Partnership / Investment Intake

Use this document as the standard structure when someone shows interest in collaboration, partnership, or investment.

## First Reply Checklist

Ask for:

- Name or organization
- GitHub username or contact path
- Interest type: investor, product partner, dive shop, creator, technical collaborator, other
- What problem they want this project to solve
- Whether they need open-source use, paid service, private deployment, or custom workflow
- Timeline
- Region or market
- Any non-confidential requirements

Do not ask them to post:

- Passwords
- API keys
- Private customer data
- Private videos
- Financial documents
- Legal documents

## Qualification Questions

1. Are you interested in funding, product partnership, customer pilot, or technical collaboration?
2. What underwater media workflow do you currently use?
3. What is the biggest current pain: speed, color quality, batch processing, DaVinci handoff, AI masking, or client delivery?
4. Do you need cloud processing, local processing, or both?
5. What result would make this useful enough to pay for?
6. Are you comfortable starting with an MVP pilot?

## Response Template

```text
Thanks for the interest in AI Color Workspace.

Current status: the repository is an open-source V2 foundation, not yet a finished commercial product.

The strongest near-term direction is an underwater AI color workflow that combines:
- scene analysis
- grading plan
- DCTL/LUT parameter engine
- scoring
- memory/learning loop
- future segmentation and DaVinci handoff

If you are open to it, please share:
1. Your role or organization
2. What type of interest this is: investment, partnership, pilot customer, or technical collaboration
3. Your target use case
4. Whether you need cloud, local, or hybrid processing
5. What success would look like for you
```

## Internal Follow-Up Notes

Track:

- Source of interest
- Date
- User type
- Seriousness level
- Required feature
- Commercial potential
- Follow-up action
