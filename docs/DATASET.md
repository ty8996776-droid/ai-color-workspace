# Dataset Plan

Current state: dataset folder exists, but no curated training or evaluation dataset has been added.

Recommended first dataset:

| Type | Count | Notes |
|---|---:|---|
| Blue water | 20 | clear ocean, diver, fish, reef |
| Green/murky water | 20 | shallow, low visibility, algae or river scenes |
| Low light | 15 | night dive, deep water, noisy clips |
| Portrait/diver | 15 | exposed skin, wetsuit, white balance risk |
| Sand/highlight | 15 | bright sand, sun shafts, surface reflection |
| Coral/fish | 15 | saturation and local color protection |

Each item should record:

- input file
- scene tags
- current workbench score
- V2 score
- human note
- failure reason

No user media is copied by this MVP.
